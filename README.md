# Quiz-app
It is a web-based skill assessment quiz application that evaluates candidates using scenario-based, role-specific questions. It features user authentication, timed quizzes, automated scoring with qualification logic, profile-based assessments, an admin dashboard, and a clean, responsive UI built using HTML, CSS, and JavaScript.
